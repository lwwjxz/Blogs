1. 根据登录和非登陆，交互和非交互shell加载的环境变量不同。   
1. 赋值语句等号两边不能有空格。  
1. 逻辑运算  
  > 整数比较  
  
  ```
  -eq 等于,如:if [ "$a" -eq "$b" ]   
  -ne 不等于,如:if [ "$a" -ne "$b" ]   
  -gt 大于,如:if [ "$a" -gt "$b" ]   
  -ge 大于等于,如:if [ "$a" -ge "$b" ]   
  -lt 小于,如:if [ "$a" -lt "$b" ]   
  -le 小于等于,如:if [ "$a" -le "$b" ]   
  <   小于(需要双括号),如:(("$a" < "$b"))   
  <=  小于等于(需要双括号),如:(("$a" <= "$b"))   
  >   大于(需要双括号),如:(("$a" > "$b"))   
  >=  大于等于(需要双括号),如:(("$a" >= "$b"))   
  
  ```
  > 字符串比较   
  
  ```
  = 等于,如:if [ "$a" = "$b" ]   
  == 等于,如:if [ "$a" == "$b" ],与=等价 
  ```
